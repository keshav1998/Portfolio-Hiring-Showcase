Automate the process of creating founder profiles on She Raises Capital's platform. This tool takes in pitch decks, websites, financials, and other key documents, and creates a comprehensive user profile without the need for manual input.

###### Automated Input
Takes in key documents and creates a user profile automatically

###### Streamlined Process
Reduces the need for manual input, saving time and effort

###### Advanced Tools
Utilizes Langchain, GPT-3.5-Instruct, and AWS Lambda for implementation

###### Customizable Profiles
Creates profiles that can be customized to suit the platform's needs

###### Efficient Overview
Get a clear overview of all your profiles in one place
